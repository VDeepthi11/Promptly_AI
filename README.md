# 🗄️ PromptVault

**PromptVault** is a lightweight Streamlit app that lets you save, organize, search, and manage AI prompts and responses in one place. It uses SQLite for storage and provides a clean interface for quick access and filtering.

---

## ✨ Features

* Save prompts with responses
* Categorize entries
* Track AI model used
* Search prompts instantly
* Filter by category
* Delete entries easily

---

## 🛠 Tech Stack

**Streamlit • SQLite • Pandas**

---

## 🚀 Run Locally

```bash
pip install streamlit pandas
streamlit run app.py
```

---

## 📂 Database Schema

`id | category | prompt | model | response`
